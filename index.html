Entregas anteriores
  1. Función ingresarTexto() permite ingresar un texto al html desde consola
  2. Función cargaTablaDeProductos carga los productos desde JS -
  3. Se agregó la linea generadorAutomatico() para cargar la lista de productos al iniciar la página.
  4. Función buscarProducto(): pide al usuario un producto por nombre e itera sobre el array 'productos' utilizando 'filter' e 'includes' para devoler todos los productos que contengan lo ingresado.
  5. Función calculaCompra(): solicita código del producto y las cantidades, se va mostrando cada producto elegido, la cantidad y el subtotal por producto. Al ingresar '0' como código de producto se finaliza la compra y se muestra el importe total. Si el usuario ingresa un código que no existe se muestra un 'alert' y vuelve a pedir otro código.
  6. Se agregó una tabla de los productos de forma estática en la página principal.
  7. Se incorpora un array de objetos llamado productos.
  8. Con la función generadorAutomatico() se cargan los productos teniendo como constructor la clase Producto.
  9. Producto incorpora dos métodos, precioConIva para calcular el precio final y restaStock para descontar unidades del stock.
  10. La función agregarProductos() permite agregar mas productos en la tabla.
  11. La función calculaCuotas() calcula el importe a pagar sobre un valor ya definido consulta al usuario en cuantas cuotas quiere abonar, e invocando a la función calculaInteres(), devuelve el importe total , las cuotas a pagar y el monto de cada una.
