Productos Naturales

Primera preentrega

  1. Se agregó la linea generadorAutomatico() para cargar automaticamente la lista de productos al iniciar la página.
  2. Función buscarProducto(): pide al usuario un producto por nombre e itera sobre el array productos utilizando filter e includes para devoler todos los productos que contengan lo ingresado.
  3. Función calculaCompra(): solicita los códigos de los productos y las cantidades, se van mostrando los productos con la cantidades y subtotal por producto. Al ingresar '0' como codigo de producto se finaliza la compra y se muestra el importe total. En este caso se utilizé 'find' ya que los codigos no se repiten. Si el usuario ingresa un código que no existe se muestra un 'alert' y vuelve a pedir otro código.
  4. Se agregó una tabla de forma estática en la página principal para poder los productos.

Entregas anteriores

  1. Se incorpora un array de objetos llamado productos.
  2. Con la función generadorAutomatico() se cargan los productos teniendo como constructor la clase Producto.
  3. Producto incorpora dos métodos, precioConIva para calcular el precio final y restaStock para descontar unidades del stock.
  4. ` La función agregarProductos() permite agregar mas productos en la tabla.
  5. La función calculaCuotas() calcula el importe a pagar sobre un valor ya definido consulta al usuario en cuantas cuotas quiere abonar, e invocando a la función calculaInteres(), devuelve el importe total , las cuotas a pagar y el monto de cada una.
